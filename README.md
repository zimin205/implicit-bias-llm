# implicit-bias-llm

이 레포지토리는 대규모 언어 모델(LLM)의 암묵적 편향을 측정하고,
이를 완화하기 위한 구조적 방법론을 연구하기 위한 실험 코드와 분석 결과를 포함한다.

본 연구는 명시적 혐오 표현이나 차별 발화를 탐지하는 것이 아니라,
겉보기에는 중립적인 문맥에서 모델이 보이는
확률적 선택, 책임 귀속, 집단 일반화 등의 암묵적 편향에 초점을 둔다.

---

## 연구 배경

최근 LLM들은 명시적인 편향 표현에 대해서는 비교적 잘 회피하도록 정렬(alignment)되어 있다.
그러나 다음과 같은 상황에서는 여전히 암묵적 편향이 관찰된다.

- 정보가 불충분한 질문에서 특정 집단을 더 그럴듯한 원인으로 선택하는 경우
- "모르겠다"는 선택지가 있음에도 불구하고 고정관념적 답변을 선호하는 경우
- 비교 구조나 서술 방식에서 특정 집단을 일관되게 불리하게 배치하는 경우

이 레포지토리는 이러한 암묵적·문맥적 편향을
모델의 최종 출력뿐 아니라 선택 선호 및 생성 과정의 관점에서 분석한다.

---

## 연구 목표

본 프로젝트의 주요 목표는 다음과 같다.

1. 암묵적 편향의 베이스라인 정의
   - 편향 완화 개입 없이(raw LLM) 모델이 보이는 자연스러운 선택 선호를 측정
   - 텍스트 출력이 아닌, 선택 구조 및 확률 분포 수준에서의 편향을 분석

2. 편향 감지–분석–수정의 역할 분리 구조 설계
   - 임베딩 기반 신호와 LLM 기반 의미 판단을 결합한 편향 감지
   - 추론 중심(reasoning-first)의 편향 분석 단계 설계

3. 설명 가능한 편향 완화 프레임워크 제안
   - 단순 필터링이 아닌, 편향의 논리적 구조(일반화, 책임 귀속 등)를 분석 대상으로 설정
   - 수정 결과뿐 아니라 수정의 근거를 함께 제공하는 구조 지향

---

## 사용 데이터셋

본 연구는 주로 다음과 같은 암묵적 편향 중심 데이터셋을 사용한다.

- BBQ (Bias Benchmark for QA)  
  정보가 불충분한 질문 상황에서 모델의 고정관념적 선택을 분석하기 위한 QA 벤치마크

- StereoSet (보조 실험)  
  집단–속성 간 암묵적 연결을 평가하기 위한 문장 수준 데이터셋
